webscanner
==========

###Descriptioin

A simple website scanning tool.

###How to install

```
	make
	make clean
```

###How to use

1. Scanning a website using the default dictionary
```
./webscanner -url [url]
```
2. Scanning a website using a designated dictionary
```
./webscanner -url [url] -dict [filepath of the dictionary]
```
